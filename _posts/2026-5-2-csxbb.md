# 一、数据来源

女权主义者口嗨很容易，然而查询数据和处理数据有一定的难度，尤其是多国数据的分析。然而我还是在世界银行的网站上同时找到了[谋杀犯罪率](https://data.worldbank.org.cn/indicator/VC.IHR.PSRC.P5)和[出生性别比](https://api.worldbank.org/v2/zh/indicator/SP.POP.BRTH.MF?downloadformat=excel)的数据。抽取国家和地区的数量N=60。

# 二、数据处理

我处理了两个核心变量，进行单变量线性回归分析，其中

1. 自变量——出生性别比
2. 因变量——犯罪率

同时我们使用Python脚本计算皮尔逊相关系数*r*以及决定系数R² 。

# 三、结果

统计结果显示，两者之间的线性关系非常弱（r=0.262），**只能解释谋杀犯罪率变化中 6.9% 的部分（R²=0.0069）**，剩下的 93.1% 的波动，无法用出生性别比来解释，只能用其他因素（如贫富差距、毒品、治安力度、战争、文化差异等）或纯粹的随机误差来解释。在实际意义中，可以忽略这种微弱的相关性，不能做出有意义的推断。

import pandas as pd
import matplotlib.pyplot as plt
import numpy as np
import statsmodels.api as sm

file_path = "国家性别比与谋杀犯罪率统计_updated.xlsx"

mask = ~(np.isnan(x) | np.isnan(y))
x_clean = x[mask]
y_clean = y[mask]
X = sm.add_constant(x_clean)  
model = sm.OLS(y_clean, X).fit()
x_plot = np.linspace(min(x_clean), max(x_clean), 100)

predictions = model.get_prediction(X_plot)
pred_frame = predictions.summary_frame(alpha=0.05) 

print(f"皮尔逊相关系数 r = {np.sqrt(model.rsquared):.3f}")
print(f"决定系数 R² = {model.rsquared:.3f}")
print(f"回归系数 (斜率): {model.params[1]:.4f}")

# 四、结语

综合以上对60个国家和地区数据的分析，我们可以得出一个清晰且有力的结论：「出生性别比」与「谋杀犯罪率」之间，不存在统计学上具有实际意义的依存关系。

具体来看，两者之间的皮尔逊相关系数仅为r=0.262，属于极弱相关。决定系数R²=0.069更是直白地宣告：「出生性别比」这一变量，最多只能解释「谋杀犯罪率」中6.9%的极小部分。这意味着，在每100个影响谋杀率变动的因素里，有超过93个是性别比之外的东西——比如社会经济水平、枪支管控政策、毒品贸易状况、治安投入、甚至是战争与民族冲突。将这些复杂的变量强行归结于「性别比失常」非蠢极坏。

**综上所述，基于目前的世界银行公开数据，没有任何证据支持「性别比失常会导致犯罪率上升」的论断。**
